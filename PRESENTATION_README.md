## 📁 HTML Slide Files

Each slide is an individual HTML file that can be:
- Viewed in a web browser
- Converted to PDF using browser print
- Converted to PPTX using online tools

## 🚀 How to Use

### View in Browser
```bash
open title_slide.html
```

### Convert to PDF
1. Open HTML in browser
2. Press Ctrl+P (or Cmd+P)
3. Select "Save as PDF"

### Convert to PPTX
Use online tools like CloudConvert or Zamzar to convert PDF to PPTX.

## 📝 Customization

Edit HTML files to customize:
- Text content
- Colors (hex codes in CSS)
- Layout and spacing
- Add images
