Overcoming Technical Challenges
Challenge 1: Model Generalization
❌ Problem
Custom CNN trained only on lab images failed on real-world photos
✓ Solution
MobileNetV2 transfer learning + heavy augmentation (rotation, zoom, brightness, noise)
📊 Result
99%+ accuracy on field photos, validated on real crops
Challenge 2: Production Deployment
❌ Problem
Python not available in production container (/usr/bin/python3.11: not found)
✓ Solution
Migrated to pure Node.js ONNX Runtime (onnxruntime-node + sharp)
📊 Result
Production-ready, no Python dependency, ~50ms inference
Challenge 3: Disease Region Detection
❌ Problem
Identifying exactly which parts of leaf are infected for visual guidance
✓ Solution
HSL color segmentation + BFS flood fill + ellipse polygon fitting
📊 Result
Accurate region highlighting with animated lasso overlay
Challenge 4: Farmer Accessibility
❌ Problem
Complex UI not suitable for rural farmers with limited tech experience
✓ Solution
Bilingual interface (English/Nepali), high-contrast mode (7:1), simple workflow
📊 Result
Farmer-friendly app, WCAG AA compliant, accessible outdoors
